= Description =

Grow your business with OlyCash. Make payments with any method such as Card, PayPal, Mobile Money, Cash Codes, Crypto and more. Withdraw your sales using the OlyCash mobile app, SMS or your nearest OlyBoss (agent) location. For more information visit our website at www.olycash.com 

= Minimum Requirements =

* WordPress 5.2
* WooCommerce 3.7.1 or greater
* PHP version 5.4 or greater. PHP 7.2 or greater is recommended
* MySQL version 5.0 or greater. MySQL 5.6 or greater is recommended


= Automatic installation =

Automatic installation is the easiest option, as WordPress handles the file transfers and you don't need to leave your web browser. To perform an automatic install:
1. Log in to your WordPress dashboard.
2. Go to: Plugins > Add New.
3. Enter “OlyCash” in the Search field, and view details about its point release, the rating and description.
4. Select “Install Now” when you’re ready
